# DigiPocket_Online_Transaction_System
This is the simple API created using FastAPI, along with it, the authorization is done through JWT Tokens.

We can register the user after which we can login into the user using the email and the password, after the login, the authorization code of JWT Bearer is provided which we need to insert as the value into the Authorize button with the Lock in the above portion of the API's Swagger UI after which all the authroized end-points for the user is being able to be used.
